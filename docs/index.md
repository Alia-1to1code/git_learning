# Bienvenue sur la documentation de git_learning

Ce projet est un exercice d'apprentissage de Git et GitHub.

## À propos

Ce dépôt contient un module Python de calcul arithmétique utilisé pour
illustrer les concepts de :

- Gestion de versions avec Git
- Collaboration via GitHub
- Intégration continue avec GitHub Actions
- Documentation automatisée avec MkDocs

## Démarrage rapide

```bash
git clone https://github.com/Alia-1to1code/git_learning.git
cd git_learning
pip install -r requirements-dev.txt
pytest tests/
``` 

!!! tip "Conseil"
    Commencez par lire le [Guide d'installation](guide/installation.md) si c'est votre première utilisation.
